Funktionen Short IDs:
1 - alle Hara Items an Spieler
2 - item an Hara geben              -> 3 Parameter: Item, Chapter und Dialog
3 - item ins Inventar einfügen      -> 4 Parameter: Item, Visibility, Chapter und Dialog
4 - item entfernen                  -> item
5 - itemsichtbarkeit ändern         -> sichtbarkeit
6 - "reset game"                    -> 3 Paramter: 0, 0 und true
7 - fortschritt in Playerdate       -> 2 Parameter: Chapter und Dialog


Wenn ein logisch richtiger aber für den block unbekannter befehl eingegbene wird, wird auf einen standartblock zurückgegriffen.
